# API-WhatsApp

Enviar mensajes a WhatsApp a traves de una API.

# Comenzar
Clonar el repositorio
```
git clone https://github.com/jesusaraujo06/api-whatsapp-ts.git
```

Entrar a la carpeta del proyecto y ejecutar en la consola:
```
npm install

npm run build

npm run dev
```
